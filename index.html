1. ¿A qué se denomina Multimedia, Hipermedia e Hipertexto?

Multimedia: es la integración de varios tipos de medios (texto, imágenes, sonido, video, animaciones) en una misma aplicación o entorno interactivo.

Hipermedia: es la evolución de la multimedia que permite la interactividad, enlazando diferentes tipos de información mediante vínculos (links).

Hipertexto: es un texto con enlaces a otros textos o recursos que permiten una lectura no lineal.

2. ¿Qué es una Página Web? ¿Y qué editores de texto se usan para crear las páginas web?

Una página web es un documento electrónico escrito en lenguaje HTML que se visualiza mediante un navegador. Puede incluir imágenes, enlaces, videos y otros elementos interactivos.

Algunos editores de texto usados para crear páginas web son:

  • Visual Studio Code
  • Sublime Text
  • Atom
  • Notepad++
  • FrontPage (clásico)

3. ¿Qué significa HTML y para qué se utiliza?

HTML significa HyperText Markup Language (Lenguaje de Marcado de Hipertexto).

Se utiliza para estructurar el contenido de una página web, mediante etiquetas que indican al navegador cómo mostrar cada elemento.

4. ¿Qué etiquetas marcan el inicio y el final de un documento HTML?

<html> ... </html>

Estas etiquetas indican el inicio y el fin del código HTML de la página.

5. Las etiquetas <BODY> y </BODY>, ¿qué indican dentro del documento HTML?

El contenido que se encuentra entre <body> y </body> representa todo lo que el usuario verá en la página web: texto, imágenes, enlaces, tablas, etc.

6. ¿Cuáles etiquetas se han declarado dentro de <HEAD> y </HEAD>?

Dentro del <head> se declaran etiquetas como:

  • <title> — título que aparece en la pestaña del navegador.
  • <meta> — metadatos sobre el documento (autor, codificación, descripción, etc.).
  • <link> — enlaza hojas de estilo CSS.
  • <script> — enlaza archivos o código JavaScript.
  • <style> — define estilos CSS directamente.

7. ¿Cuáles son las etiquetas de los atributos de texto? Realizar un ejemplo con cada uno de ellos.

  • <b> → Texto en negrita
  • <i> → Texto en cursiva
  • <u> → Texto subrayado
  • <small> → Texto pequeño
  • <mark> → Texto resaltado
  • <sup> → Textosuperíndice
  • <sub> → Textosubíndice

8. ¿Cuál es la etiqueta que participa como parámetro en otras etiquetas y qué función cumple?

El atributo style participa como parámetro en muchas etiquetas y se usa para aplicar estilos directamente.

<p style="color:blue;">Texto azul</p>

9. ¿Cuáles son las formas de utilizar las etiquetas de listas? Describir cada una a través de ejemplos reales.

Listas ordenadas:


<ol>
    <li>Primero</li>
    <li>Segundo</li>
</ol>

Listas desordenadas:


<ul>
    <li>HTML</li>
    <li>CSS</li>
    <li>JavaScript</li>
</ul>

Listas de definición:


<dl>
    <dt>HTML</dt>
    <dd>Lenguaje de marcado</dd>
</dl>

10. ¿Cuál sería la forma de colocar un color de fondo a la página web?

Se utiliza la propiedad CSS background-color.


body {
    background-color: lightblue;
}

11. ¿Para qué se utiliza la etiqueta <FONT> en la página web? Aplicar un ejemplo.

La etiqueta <font> se utilizaba para cambiar color, tamaño y tipo de letra del texto, aunque hoy está obsoleta.

<font color="red" size="5">Texto rojo</font>

Se reemplaza por CSS:

p { color: red; font-size: 20px; }

12. ¿Qué diferencia hay en utilizar la etiqueta <P> o la etiqueta <DIV>?

<p> se usa para crear párrafos de texto.

<div> se usa como contenedor genérico que puede agrupar texto, imágenes u otros elementos. No tiene un significado semántico propio.

13. ¿Cuál es la estructura básica del código HTML?


<!DOCTYPE html>
<html>
<head>
    <title>Mi página</title>
</head>
<body>
    <h1>Hola mundo</h1>
</body>
</html>

14. ¿Cuáles son las etiquetas de encabezamiento o título? Explicar cada una por medio de ejemplos.

Las etiquetas de encabezado van desde <h1> hasta <h6> y se usan para jerarquizar los títulos.


<h1>Título principal</h1>
<h2>Subtítulo</h2>
<h3>Encabezado menor</h3>
